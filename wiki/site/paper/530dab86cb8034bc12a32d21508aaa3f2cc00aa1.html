LeWorldModel: Stable End-to-End Joint-Embedding Predictive Architecture from Pixels

LeWorldModel (LeWM) is a world model architecture introduced by Maes, Le Lidec, Scieur, LeCun, and Balestriero (2026) that trains a Joint-Embedding Predictive Architecture (JEPA) stably end-to-end from raw pixels using only two loss terms: a next-embedding prediction loss and SIGReg, a regularizer enforcing Gaussian-distributed latent embeddings. LeWM is the first end-to-end JEPA to avoid representation collapse without stop-gradients, exponential moving averages, pre-trained encoders, or auxiliary supervision. With only 15M parameters trainable on a single GPU in a few hours, LeWM plans up to 48× faster than foundation-model-based world models while achieving competitive control performance across diverse 2D and 3D tasks.

Representation Collapse in JEPA World Models

Joint-Embedding Predictive Architectures (JEPAs) learn world models by encoding observations into compact latent spaces and training a predictor to model temporal dynamics—predicting the next latent state given the current state and action. This framework is appealing because it focuses on learning only the most predictive features of the environment, discarding irrelevant details.

However, JEPAs are highly susceptible to representation collapse: the encoder and predictor can trivially satisfy the prediction objective by mapping all observations to a constant representation. If all latent states are identical, the predictor trivially achieves zero prediction error regardless of the action taken. Detecting and preventing this failure mode is the central challenge in JEPA training.

Existing methods employ various anti-collapse strategies, each with significant drawbacks. Stop-gradient operators (used in SimSiam, BYOL) prevent gradient flow through one branch of the network but introduce instability and require careful tuning of the stopped branch's learning rate. Exponential moving average (EMA) target encoders (used in BYOL, I-JEPA) decouple the target from the online network but add complexity and another hyperparameter. Pre-trained frozen encoders (used in DINO-WM) prevent collapse entirely but sacrifice end-to-end learning and remain bounded by the pretraining knowledge.

LeWM introduces a principled alternative: a regularization term SIGReg that enforces a Gaussian distribution on latent embeddings. If embeddings follow a non-degenerate isotropic Gaussian, they cannot collapse to a constant—the Gaussian has full rank support covering the entire embedding space. SIGReg provides a formal anti-collapse guarantee with a single tunable hyperparameter, versus six hyperparameters in the prior best end-to-end alternative (PLDM).

SIGReg: Gaussian Regularization via Random Projections

SIGReg (Statistical Isotropic Gaussian Regularization) enforces that the distribution of latent embeddings matches an isotropic Gaussian by testing normality along random univariate directions. The key insight is that a multivariate distribution is Gaussian if and only if all its univariate projections are Gaussian (the Cramér-Wold theorem). Testing normality along multiple random directions therefore tests for full Gaussian structure.

For each random direction v ~ Uniform(S^{d-1}), the scalar projections {v^T zᵢ} of all latent embeddings {zᵢ} in a batch should follow a standard normal distribution N(0,1) if the embeddings are Gaussian. SIGReg computes a normality test statistic (such as a Kolmogorov-Smirnov or Anderson-Darling test) for these scalar projections and includes the test statistic as a penalty in the training loss.

Aggregating normality statistics over multiple random directions encourages the full embedding distribution to match an isotropic Gaussian. The single hyperparameter λ controls the regularization strength: larger λ enforces Gaussianity more strictly but may prevent the encoder from learning discriminative features; smaller λ provides weaker collapse resistance. The paper shows that λ can be found via logarithmic search (trying λ ∈ {0.01, 0.1, 1.0}), making hyperparameter tuning tractable.

A formal theorem proves that if SIGReg achieves its minimum value (i.e., projections are perfectly Gaussian), the latent representations cannot be collapsed: the embedding distribution has full support in ℝᵈ and positive entropy in every direction. This provides a stronger anti-collapse guarantee than heuristic methods like stop-gradient or EMA, which prevent collapse empirically but lack formal guarantees.

End-to-End Architecture and Training

Figure 1: LeWorldModel Latent Planning. Given an initial observation o1 and a goal og, the world model

LeWM consists of two learnable components trained jointly end-to-end: a convolutional or Transformer encoder mapping raw pixel observations to compact latent vectors, and a recurrent predictor (typically a GRU or small Transformer) modeling temporal dynamics by predicting zₜ₊₁ from zₜ and action aₜ.

The training objective has exactly two terms: L = L_pred + λ · L_SIGReg. The prediction loss L_pred is mean-squared error between the predictor's output ẑₜ₊₁ and the target zₜ₊₁ computed by the encoder from the next observation. This is the standard JEPA prediction objective. The regularization term L_SIGReg penalizes non-Gaussianity of the encoder outputs.

Critically, both encoder and predictor are trained with standard backpropagation without any stop-gradient operators, momentum encoders, or frozen components. The SIGReg term provides sufficient regularization to prevent collapse while allowing both components to adapt. This simplicity enables the system to train stably on a single GPU, lowering the barrier for research experimentation.

The architecture uses compact encoders (15M parameters total) designed for speed rather than expressive power. Input observations are encoded to 16-64 dimensional latent vectors (compared to 768+ for vision foundation models like DINO), enabling extremely fast latent-space planning. The small latent space also reduces the burden on the predictor, which only needs to model dynamics in 16–64 dimensions.

Latent-Space Planning and Physical Understanding

LeWM supports model-predictive control (MPC) in latent space: given a current observation, the encoder produces a starting latent state, and the predictor simulates trajectories by autoregressively applying the dynamics model for different action sequences. The action sequence with the best predicted outcome (according to a task-specific reward or goal criterion) is executed.

Because LeWM's latent space is compact (16–64 dimensions versus 196+ for DINO-WM patches), encoding observations and simulating trajectories is dramatically faster. LeWM plans up to 48× faster than DINO-WM under the same computational budget, achieving higher control success rates in the same wall-clock time. On PushT, LeWM achieves 90% success versus DINO-WM's 13% at fixed FLOPs.

Beyond control, the paper probes whether LeWM's latent space encodes meaningful physical structure. Linear probing experiments test whether physical quantities (object position, velocity, contact forces) can be decoded from the latent state using simple linear regressors. LeWM's latents encode these quantities with high accuracy, suggesting the encoder learns physics-relevant abstractions rather than arbitrary compression.

Surprise detection experiments test whether LeWM assigns low prediction confidence to physically implausible events (e.g., objects passing through walls, sudden position teleportation). The predictor's MSE prediction error serves as a surprise signal: physically plausible transitions are predicted accurately; implausible ones produce high prediction error. This emergent capability suggests LeWM learns a genuine physics model rather than a shallow action-to-observation mapping.

Comparison with Existing World Model Methods

Figure 2: Predictor rollouts on PushT and OGBench-Cube. We visualize decoded latent plans

DINO-WM (a JEPA world model using frozen DINO features) achieves strong performance but requires a pre-trained foundation encoder, making it dependent on the pretraining knowledge and computationally expensive per inference step. LeWM surpasses DINO-WM on PushT and OGBench-Cube under fixed-compute budgets by being 48× faster per planning step.

PLDM (the only prior end-to-end JEPA world model) requires six loss hyperparameters and complex multi-term objectives for anti-collapse, making training unstable and hyperparameter search expensive. LeWM reduces this to one hyperparameter (λ for SIGReg) with provable anti-collapse guarantees, enabling logarithmic hyperparameter search rather than exhaustive grid search.

Task-specific world models (Dreamer, TD-MPC) achieve strong performance but require reward signals or privileged state access during training, limiting their applicability to tasks with available reward functions. LeWM is reward-free and observation-only, learning from raw pixel trajectories without any task-specific supervision.

Across 2D manipulation (PushT, Relay Kitchen), 3D navigation (OGBench), and locomotion (DMControl) tasks, LeWM achieves competitive or superior performance while requiring only a single GPU, no pre-trained components, and no reward supervision. This combination of simplicity, efficiency, and generality represents a substantial advance in the practicality of world model learning.

Concept Breakdown

SIGReg (Statistical Isotropic Gaussian Regularization)

A regularization term enforcing Gaussian-distributed latent embeddings by testing normality along multiple random univariate projections. By the Cramér-Wold theorem, matching Gaussianity along all directions ensures the full multivariate distribution is Gaussian, providing a formal anti-collapse guarantee with a single hyperparameter λ.

End-to-End Training Without Heuristics

LeWM trains encoder and predictor jointly using standard backpropagation without stop-gradients, EMA encoders, frozen components, or auxiliary losses beyond SIGReg. This simplicity enables training on a single GPU and reduces training instabilities inherent to methods with multiple interacting components and objectives.

Compact Latent World Model

LeWM encodes observations to 16–64 dimensional latent vectors, far smaller than foundation model patch representations (768+ dimensions for DINO). This compact representation enables extremely fast latent-space planning (48× faster than DINO-WM) while maintaining sufficient information for accurate dynamics modeling.

Autoregressive Latent Dynamics Predictor

A recurrent network (GRU or small Transformer) that autoregressively predicts the next latent state given the current latent state and action: ẑₜ₊₁ = f(zₜ, aₜ). This predictor is trained end-to-end with the encoder, learning to model environment dynamics in the compact latent space induced by SIGReg regularization.

Model-Predictive Control in Latent Space

At inference time, LeWM supports planning by simulating multiple action sequences forward in latent space using the dynamics predictor, then executing the sequence achieving the best predicted outcome. The compact latent space makes this planning dramatically faster than pixel-space simulation or foundation-model-based prediction.

Physical Structure Probing

Evaluation methodology testing whether LeWM's latent space encodes physically meaningful quantities (position, velocity, contact forces) via linear probing. High linear decodability of physical quantities indicates the encoder learned physics-relevant abstractions. Complement to control performance, demonstrating the latent space's interpretability.

Mathematics

LeWM two-term training objective

$$\mathcal{L} = \underbrace{\|f_\theta(z_t, a_t) - z_{t+1}\|_2^2}_{\text{prediction loss}} + \lambda \cdot \underbrace{\mathcal{L}_{\text{SIGReg}}(\{z_t\})}_{\text{Gaussian regularization}}$$
The total loss has exactly two terms. The prediction loss trains the dynamics predictor to accurately forecast the next latent state from the current state and action. The SIGReg term ensures the encoder does not collapse all states to a constant by penalizing deviations from an isotropic Gaussian distribution. λ is the single hyperparameter controlling the anti-collapse strength.
SymbolMeaning
$f_\theta$dynamics predictor network
$z_t$latent embedding of observation at time t
$a_t$action taken at time t
$z_{t+1}$target latent embedding from encoder at t+1
$\lambda$regularization strength (single hyperparameter)

SIGReg regularization via random projection normality tests

$$\mathcal{L}_{\text{SIGReg}} = \frac{1}{K}\sum_{k=1}^K T_{\text{norm}}\!\left(\{v_k^\top z_i\}_{i=1}^N\right), \quad v_k \sim \text{Uniform}(S^{d-1})$$
For K random unit vectors v₁,...,vₖ on the sphere, project all N latent vectors zᵢ onto each direction to get scalar values {vₖ^T zᵢ}. A normality test statistic T_norm (e.g., Anderson-Darling) measures deviation of these scalars from N(0,1). Averaging over K directions tests for global Gaussianity. By Cramér-Wold theorem, matching all univariate projections ensures multivariate Gaussianity.
SymbolMeaning
$K$number of random projection directions
$v_k$k-th random unit vector on sphere
$T_{\text{norm}}$univariate normality test statistic
$N$batch size
$S^{d-1}$unit sphere in d dimensions

End-to-end training update

$$z_{t+1} = g_\phi(o_{t+1}), \quad \hat{z}_{t+1} = f_\theta(z_t, a_t), \quad \theta, \phi \leftarrow \nabla(\mathcal{L})$$
Both the encoder g_φ and predictor f_θ are updated simultaneously via backpropagation through the same loss. There is no stop-gradient, no EMA, and no frozen component. z_{t+1} is computed by the encoder from the actual next observation, and ẑ_{t+1} is the predictor's estimate. The prediction error (MSE) plus SIGReg flows gradients through both components.
SymbolMeaning
$g_\phi$encoder network with parameters φ
$f_\theta$predictor network with parameters θ
$o_{t+1}$raw pixel observation at t+1
$\nabla(\mathcal{L})$gradient of total loss with respect to all parameters
