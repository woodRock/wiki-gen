    PolarQuant: Quantizing KV Caches with Polar Transformation

    **Authors:** Insu Han, Praneeth Kacham, Amin Karbasi, Vahab Mirrokni, Amir Zandieh **Year:** 2025 **Venue:** arXiv (cs.LG) **Paper ID:** ef9485a2522f64bca0f5cf67edc28a11984790e8

    PolarQuant is a specialized quantization framework designed to compress the Key-Value (KV) cache in Large Language Models (LLMs). By representing embedding vectors in polar coordinates rather than traditional Cartesian coordinates, PolarQuant achieves high compression ratios (up to 4.2x) with minimal loss in model accuracy. The method leverages random preconditioning to induce a predictable distribution of angles, eliminating the need for expensive per-block normalization.

    Summary

    The KV Cache Bottleneck

    In autoregressive Transformers, the KV cache stores key and value embeddings for all previously generated tokens to avoid redundant computations. As context length increases, the memory demand for these caches scales linearly, often becoming the primary bottleneck for serving long-sequence models.

    Polar Transformation Heuristic

    Traditional quantization methods (like 4-bit or 8-bit integer quantization) operate on Cartesian coordinates and require storing scale and zero-point metadata for each block of numbers. PolarQuant shifts this paradigm by converting vectors into a Recursive Polar Transformation.

    For a vector \(x \in \mathbb{R}^d\), the transformation groups coordinates into pairs and recursively computes radii and angles: $\(\psi^{(1)}_j = \tan^{-1}(x_{2j}/x_{2j-1})\)$ $\(r^{(1)}_j = \sqrt{x_{2j-1}^2 + x_{2j}^2}\)$

    This process is repeated \(\log_2 d\) times, resulting in a single final radius and a set of angles.

    Random Preconditioning

    A core contribution of PolarQuant is the use of Random Preconditioning (multiplying the input by a random rotation matrix \(S\)). According to the Johnson-Lindenstrauss lemma, this preserves the inner product while ensuring the resulting vectors follow a multivariate normal distribution.

    Crucially, after preconditioning, the angles \(\psi\) exhibit a tightly bounded and highly concentrated distribution (Lemma 1). This concentration allows for the design of optimal, data-oblivious codebooks, removing the overhead of storing per-block normalization constants.

    Glossary

    Term Definition
    KV Cache A memory buffer used in Transformers to store Key and Value vectors of previous tokens to accelerate inference.
    Quantization The process of mapping continuous or high-precision values to a finite set of discrete symbols to reduce memory.
    Random Preconditioning Applying a random rotation matrix to input data to normalize its distribution while preserving geometric properties.
    Polar Coordinates A 2D coordinate system where each point is determined by a distance from a reference point and an angle from a reference direction.
    Data-Oblivious Algorithms that do not require access to the specific input distribution to maintain performance.
