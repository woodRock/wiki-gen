## Overview
The dominant sequence transduction models have historically been based on complex recurrent or convolutional neural networks. This paper proposes the Transformer, a novel network architecture based solely on attention mechanisms, dispensing with recurrence and convolutions entirely. The model proves to be highly parallelizable, significantly faster to train, and achieves state-of-the-art results on machine translation tasks.

## Summary
The core of the Transformer architecture is its reliance on the **Self-Attention** mechanism to draw global dependencies between input and output. The most critical component introduced is **Scaled Dot-Product Attention**, defined as:

$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

To allow the model to jointly attend to information from different representation subspaces at different positions, the authors introduce **Multi-Head Attention**:

$$\text{MultiHead}(Q, K, V) = \text{Concat}(\text{head}_1, ..., \text{head}_h)W^O$$

where $\text{head}_i = \text{Attention}(QW_i^Q, KW_i^K, VW_i^V)$.

The architecture features an encoder-decoder structure. Since it lacks recurrence or convolutions, **Positional Encodings** (using sine and cosine functions of different frequencies) are injected into the input embeddings to provide information about the relative or absolute position of tokens in the sequence. The Transformer achieves significant improvements over RNN-based encoder-decoder architectures, requiring far fewer sequential operations and exhibiting a shorter maximum path length for learning long-range dependencies.

## Glossary
| Term | Definition |
| :--- | :--- |
| **Transformer** | A sequence transduction model based entirely on attention mechanisms, avoiding recurrent layers. |
| **Self-Attention** | An attention mechanism relating different positions of a single sequence to compute its representation. |
| **Scaled Dot-Product Attention** | An attention function that scales the dot product of queries and keys by the inverse square root of the key dimension to prevent small gradients in the softmax. |
| **Multi-Head Attention** | A mechanism that runs several attention layers in parallel, allowing the model to attend to different representation subspaces. |
| **Positional Encoding** | Fixed or learned representations added to input embeddings to inject sequence order information into models lacking recurrence. |
